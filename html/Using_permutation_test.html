Simulate some data

mu1 = 0; sigma1 = 1; n1 = 20;
mu2 = 0.2; sigma2 = 1.3; n2 = 20;
groupA = normrnd(mu1, sigma1, [n1,1]);
groupB = normrnd(mu2, sigma2, [n2,1]);

Two-sided independent difference of means

p = permutation_test(groupA, groupB, 'vis', true)
p = 0.7318
